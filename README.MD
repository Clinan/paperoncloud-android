## 运行说明
#### 需要安装Android studio

# 项目简介
本分支是整合了AndroidUI+Unity3D，一般用于发布稳定发行版本。可以直接clone到Android Studio中运行。


2. [UnityAR](https://github.com/Clinan/PaperOnCloud)
```

```
